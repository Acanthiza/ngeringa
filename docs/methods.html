2 Methods

2.1 Surveys

Surveys occurred in the morning on one day in spring. Survey were occurred in each of 5 years: 2019, 2020, 2021, 2023 and 2024.

All individual birds were recorded directly into Avenza.

2.2 Analysis

2.3 Workflow

The data import, cleaning, analysis and report writing were done in a scripted workflow using the programs R and ‘R-studio Desktop’. R (R Core Team 2025) is an open source software environment for statistical computing and graphics. Base R can be extended via a range of open source packages to enable specific tasks or analyses. The packages used to produce this report are listed in Table 2.1.

R-studio Desktop is a set of open source tools built to facilitate interaction with R.
